## 💡 使用技巧

### 快速搜索

在项目根目录使用 grep 搜索关键词：

```bash
# 搜索所有发布文档中的关键词
grep -r "关键词" RELEASE*.md CHANGELOG.md

# 搜索技术文档
grep -r "关键词" docs/
```

### 文档导出

将文档导出为 PDF（如需要）：

```bash
# 使用 pandoc 或其他工具
pandoc RELEASE_v0.1.6_CN.md -o RELEASE_v0.1.6_CN.pdf
```
