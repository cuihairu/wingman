# Workspace Product Delivery Guide（历史资料）

本文档保留的是早期 `workspace` 产品交付方案，不代表当前 `Wingman` 项目已具备对应产品链路。

## 当前状态

以下能力在当前仓库中没有形成真实后端闭环：

- 函数目录到 Workspace 的完整交付链路
- Workspace 发布、取消发布、回滚
- 面向控制台的正式 Workspace 运行态交付

## 当前仓库真实方向

当前 `Wingman` 仓库的真实主线围绕：

- Agents
- Workflows
- Scripts
- Profile
- Audit

## 使用边界

本文档仅适用于：

- 历史方案归档
- 迁移取证
- 未来重建时的产品术语参考

本文档不适用于：

- 当前版本规划
- 当前功能承诺
- 当前联调与测试

## 结论

如需重建 Workspace 产品交付链路，应基于当前 Wingman 目标重新梳理，而不是直接采用本文档。
