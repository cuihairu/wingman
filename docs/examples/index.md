# 示例脚本

以下示例展示 Wingman 的各种功能用法：

## 基础示例

- [Hello World](/examples/hello-world) - 最简单的示例
- [像素检测](/examples/pixel-detection) - 检测屏幕上的指定颜色
- [图像匹配](/examples/image-matching) - 查找屏幕上的图像
- [自动化循环](/examples/auto-loop) - 持续监控并执行操作
- [宏录制](/examples/macro-record) - 录制并回放操作

## 运行示例

```bash
wingman.exe scripts/examples/hello-world.lua
```
