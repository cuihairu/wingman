- 第三方依赖统一由 vcpkg 管理；禁止回退系统库 / FetchContent

- 代码注释语言与代码库一致（中文）
- 不主动 git 提交/分支，除非用户明确要求
- TDD：先写失败测试 → 实现 → 通过 → 提交
