# API 参考

Wingman 提供以下 API 模块：

## 模块列表

### 核心模块

- [screen](/api/screen) - 屏幕操作相关 API
- [input](/api/input) - 输入模拟相关 API

### 网络与数据

- [http](/api/http) - HTTP 客户端 API
- [json](/api/json) - JSON 解析与序列化 API
- [kv](/api/kv) - 键值存储 API

### 高级功能

- [remote](/api/remote) - 远程控制 API
- [orchestrator](/api/team) - 组队编排 API

## 即将添加

- window - 窗口管理相关 API
- process - 进程管理相关 API
- human - 人性化模拟 API
- macro - 宏录制 API
- trigger - 触发器系统 API
- util - 工具函数 API
