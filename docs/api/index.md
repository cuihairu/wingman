# API 参考

Wingman 提供以下 API 模块：

## 模块列表

- [screen](/api/screen) - 屏幕操作相关 API
- [input](/api/input) - 输入模拟相关 API
- [window](/api/window) - 窗口管理相关 API
- [process](/api/process) - 进程管理相关 API
- [human](/api/human) - 人性化模拟 API
- [macro](/api/macro) - 宏录制 API
- [trigger](/api/trigger) - 触发器系统 API
- [util](/api/util) - 工具函数 API
- [remote](/api/remote) - 远程控制 API
- [debug](/api/debug) - 调试相关 API
